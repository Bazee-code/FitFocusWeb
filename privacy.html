Privacy Policy

1. Information Collection

1.1 Required Permissions

  • Health: Required for accessing your health statistics
  • Screen Time: To display your device usage statistics
  • Notifications: To send relevant app notifications

1.2 Analytics Data

  • App interaction events
  • Feature usage statistics
  • Duration settings changes
  • Age range (provided during onboarding)
  • Declared screen time statistics
  • User goals and preferences
  • Error events for troubleshooting

2. Data Usage and Protection

2.1 Image Processing

  • Processed in real-time using Google's Vision API
  • Not stored permanently
  • Not shared beyond the necessary API calls
  • Used solely for grass detection verification

2.2 Location Data

  • Used only for sunset time calculation
  • Processed locally on your device
  • Not stored or transmitted to any servers
  • Not shared with third parties

2.3 Screen Time Data

  • Stored locally and not transmitted
  • Free version users access current stats
  • Premium version users access history

3. Third-Party Services

3.1 Google Vision API

  • Used for grass detection
  • Subject to Google’s privacy policy

3.2 Firebase Analytics

  • Used for anonymous usage analytics
  • Subject to Google’s privacy policy
  • Collects non-personally identifiable info

4. Your Rights

  • Opt-out via device settings
  • Delete local data by uninstalling app
  • Control permissions via device settings

5. Data Retention

  • Analytics retained anonymously
  • Local data remains until uninstalled
  • Images processed real-time only

6. Changes to Privacy Policy

We will notify users of changes via app or email.

7. Contact Us

If you have any questions, email us at help@touchgrass.now.
